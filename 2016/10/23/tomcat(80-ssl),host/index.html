2.host（80)

  • 配置host

sudo vi /etc/hosts

  • 80端口
    • 1.注意网络代理
    • 2.用telnet试一下是否通
      telnet 127.0.0.1 80
      get http
      1. 注意apache占用了80端口（不报错，奇怪的很。）.
        重启apache：sudo /usr/sbin/apachectl restart
        关闭apache：sudo /usr/sbin/apachectl stop
        开启apache：sudo /usr/sbin/apachectl start

3.chorme 证书信任

  • 点击生成的*.cer。导入到chorme

  • chorme–>设置–>https/ssl 管理证书

    右键–>简介处–>更改为始终信任
